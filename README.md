# Binary to Decimal

*Binary to Decimal* is a web app that allows the user to enter strings of 0's and 1's, in any sequence and then displays its decimal equivalent.

## Built With

[![Vue-logo]][Vue-link]

## Project Setup

### Install dependencies

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
